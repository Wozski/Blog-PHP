# Blog-PHP
## 目的：使用前端技術:HTML、CSS、JS + 後端 PHP 語法、資料庫架設一個簡單的部落格。
### 功能：登入、登出、註冊、留言、編輯、預防 SQL injection、XSS 攻擊。
